## Properties

#### `fireSlot` {: .sp-string}

:   The bone of the rigged model to place the fire effect at.

#### `fireVobtreeName` {: .sp-string}

:   The name of the template file which contains the configuration of the fire effect. For example: `FIRETREE_MEDIUM.ZEN`.
